# 🌊 Save Our Seas (SOS)

> Protecting Our Waterways. Building Cleaner Communities.

Save Our Seas (SOS) is a public environmental awareness movement dedicated to reducing plastic pollution, promoting responsible waste management, and protecting rivers and waterways through education, research, advocacy, and strategic partnerships.

The initiative is executed through **Project TIDE (Transforming Infrastructure, Disposal & Environment)**, a community-driven program focused on environmental sustainability and long-term behavioral change.

---

## 🎯 Mission

To inspire individuals, schools, communities, institutions, and organizations to become active stewards of the environment by promoting proper waste disposal, environmental education, and collaborative sustainability initiatives.

---

## 🌍 Vision

To build a future where every community contributes to cleaner waterways, healthier ecosystems, and a more sustainable Nigeria and Africa.

---

## ⚡ Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/save-our-seas.git
```

Navigate into the project

```bash
cd save-our-seas
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
